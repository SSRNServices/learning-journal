# Contributing to Learning Journal

Thank you for your interest in contributing! We welcome all contributions that help improve this project.

## Getting Started

1. **Fork the Repository**: Create a personal copy of the repository on GitHub.
2. **Clone the Fork**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/learning-journal.git
   ```
3. **Create a Branch**: Use a descriptive name for your branch.
   - `feat/feature-name`
   - `fix/bug-description`
   - `chore/task-name`

## Commit Guidelines

We follow [Conventional Commits](https://www.conventionalcommits.org/) for clear and structured history:
- `feat:` for new features
- `fix:` for bug fixes
- `chore:` for maintenance or automation tasks
- `docs:` for documentation changes

Example: `chore: update daily commit schedule`

## Pull Request Process

1. Ensure your branch is up to date with the `main` branch.
2. Provide a clear description of the changes in your PR.
3. Link any related issues.
4. Once submitted, a maintainer will review your changes.

## Code of Conduct

By participating in this project, you agree to abide by our [Code of Conduct](CODE_OF_CONDUCT.md).
